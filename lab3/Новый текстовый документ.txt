1. Базовые рекомендуемые параметры (без модификации шаблонов)

# Оптимальные параметры для стандартного ACO
self.n_ants = 20          # Количество муравьев (10-50)
self.n_iterations = 200   # Количество итераций (100-500)
self.alpha = 1.0          # Влияние феромонов (обычно 1-2)
self.beta = 3.0           # Влияние эвристики (обычно 2-5)
self.evaporation_rate = 0.3  # Скорость испарения (0.1-0.5)
self.q = 100              # Константа для феромонов (50-200)

Обоснование:

Больше муравьев (n_ants) улучшает исследование пространства
Увеличенный beta (3.0) сильнее учитывает расстояния между вершинами
Умеренное испарение (0.3) балансирует между исследованием и эксплуатацией

2. Параметры с модификацией шаблонов

# Оптимальные параметры для ACO с шаблонами
self.n_ants = 15          # Меньше муравьев, т.к. шаблоны дают хорошие начальные решения
self.n_iterations = 150   # Меньше итераций благодаря шаблонам
self.alpha = 1.2          # Чуть больше влияние феромонов
self.beta = 2.5           # Чуть меньше влияние эвристики
self.evaporation_rate = 0.4  # Более быстрое испарение
self.q = 150              # Больше феромона для хороших решений

Обоснование модификаций:
Уменьшаем количество муравьев и итераций, так как шаблоны уже дают качественные начальные решения
Увеличиваем alpha для большего учета феромонов на шаблонных путях
Немного уменьшаем beta, так как шаблоны уже учитывают геометрию графа
Увеличиваем испарение для быстрейшего забывания плохих путей
Увеличиваем q для более сильного выделения хороших решений


3. Эмпирические правила выбора параметров

Размер графа:
Для графов < 20 вершин: n_ants = 10-15, iterations = 100-200
Для графов 20-50 вершин: n_ants = 15-30, iterations = 200-400
Для графов > 50 вершин: n_ants = 30-50, iterations = 400-1000

Плотность графа:
Для плотных графов увеличиваем beta (до 5.0)
Для разреженных графов уменьшаем beta (до 2.0)

С модификацией:

Всегда уменьшаем количество муравьев на 20-30%
Уменьшаем итерации на 25-50%
Увеличиваем alpha на 10-20%

Эти настройки обеспечивают баланс между:
Скоростью работы (меньше муравьев/итераций с шаблонами)
Качеством решений (адаптивные параметры)
Устойчивостью к различным типам графов
